🚀 如何调试 TomatoMonkey

  1. 确保浏览器已安装 Tampermonkey 扩展
  2. 点击下方按钮安装/更新用户脚本：
    安装 TomatoMonkey 脚本
  3. 刷新本页面，脚本将自动加载
  4. 使用快捷键 Ctrl+Shift+T 或点击右上角的 🍅 按钮打开设置面板
  5. 或者通过 Tampermonkey 菜单访问设置
